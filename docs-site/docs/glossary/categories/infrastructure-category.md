## Container & Orchestration

### Kubernetes (K8s)
Container orchestration platform for automating deployment, scaling, and management.

**Key Concepts**: Pods, Services, Deployments, StatefulSets, ConfigMaps

---

### Docker
Platform for developing, shipping, and running applications in containers.

**Use Cases**: Development environments, microservices packaging, CI/CD

---

### Helm
Package manager for Kubernetes applications (charts).

---

## Infrastructure as Code (IaC)

### Terraform
Infrastructure provisioning tool using declarative configuration.

**Providers**: AWS, Azure, GCP, Kubernetes

---

### CloudFormation
AWS-native infrastructure as code service.

---

## Monitoring & Observability

### Prometheus
Open-source monitoring and alerting toolkit.

**Key Features**: Time-series database, PromQL query language, alerting

---

### Grafana
Analytics and monitoring platform for visualizing metrics.

---

## CI/CD

### Continuous Integration (CI)
Practice of automatically building and testing code changes.

**Tools**: GitHub Actions, Jenkins, GitLab CI, CircleCI

---

### Continuous Deployment (CD)
Automatically deploying code changes to production.

---

## Operations

### [Runbook](../terms/runbook.md)
Operational procedures for handling incidents, deployments, and maintenance tasks.

---

### SLO (Service Level Objective)
Target level for service reliability (e.g., 99.9% uptime).

---

### SLA (Service Level Agreement)
Contract defining service guarantees with consequences for violations.
