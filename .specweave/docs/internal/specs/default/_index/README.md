## Brownfield-First Architecture

**No Duplication**:

- ✅ User stories moved to `.specweave/docs/internal/strategy/`
- ✅ Specs contain ONLY feature specs (FS-*)

**2-Letter Codes**:
- **FS** = Feature Spec (living docs) → This folder
- **PRD** = Product Requirements → `strategy/`
- **HLD** = High-Level Design → `architecture/`
- **ADR** = Architecture Decision → `architecture/adr/`
- **RUN** = Runbook → `operations/`
- **SLO** = Service Level Objective → `operations/`
- **NFR** = Non-Functional Requirement → `operations/`
- **TST** = Test Strategy → `delivery/`

**Result**: Each document type has ONE home!
