# US-003: Status Propagation

As a project manager, I want status changes in external tools to reflect in SpecWeave so that I have a single source of truth.

## Acceptance Criteria

- [ ] **AC-US3-02**: Status mapping is configurable per tool (P2, testable)
- [ ] **AC-US3-03**: Conflicting status changes trigger notifications (P2, testable)
